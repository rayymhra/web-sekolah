Contribution Guidelines

Langkah-langkah untuk berkontribusi dalam proyek ini:

Fork repository ini.

Buat branch baru untuk fitur atau perbaikan Anda:

git checkout -b nama-fitur-anda

Commit perubahan Anda:

git commit -m "Deskripsi perubahan"

Push branch ke repository Anda:

git push origin nama-fitur-anda

Buat pull request ke repository utama.
